The files contain the following:

* deut.wf : position space deuterium wave functions for Argonne V18 u(r), w(r)
* deut.wfk : position space deuterium wave function for Argonne V18 u(k), w(k)

Units : 
* u(r) and w(r) in Fermi^(-1/2) and r in Fermi
* u(k) and w(k) in Fermi^(3/2)  and k in 1/Fermi

Normalizations:
* Integral( [u(r)**2+ w(r)**2] dr) =1
* Integral( [u(k)**2+ w(k)**2] k**2 dk) =1

SOURCE:
http://www.phy.anl.gov/theory/research/av18/
